URL Shortener Web App

Home Page:
This page serves as the entry point for the URL shortener application. It includes a form where users can enter a long URL. The form submission and URL generation logic can be handled on the server side.

Shortened URL Display Page:
After a user submits a long URL, this page displays the shortened version of the URL generated by the application.

Redirect Page:
This page is responsible for redirecting users from a shortened URL to the original long URL.
